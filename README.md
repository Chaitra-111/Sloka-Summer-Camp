# Sloka Summer Camp

A Next.js and Tailwind CSS landing page for a one-week online evening sloka summer camp for kids.

## Features

- Parent-friendly class information and 7-day schedule
- Kid-focused content with teaching and sloka-themed images
- Payment details with a QR scanner placeholder
- Registration form for student, parent, phone number, doubts, and payment screenshot upload
- WhatsApp group button that unlocks only after payment proof is uploaded

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customize Before Publishing

Update these placeholders in `src/app/page.tsx` and `public/payment-qr.svg`:

- WhatsApp group invite link
- UPI ID and payee name
- Payment QR scanner
- Camp fee, timing, and contact details
